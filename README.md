# EmojiPedia 🌟

**EmojiPedia** is your ultimate emoji companion! Explore thousands of emojis, discover their meanings, and stay up-to-date with the latest emoji trends. Whether you’re texting, posting, or just curious, EmojiPedia makes it fun and easy to find the perfect emoji for every mood and moment. Dive in, express yourself, and never miss a smile, wink, or heart again! 💖✨

## Features

- 🔍 **Search & Discover:** Quickly find emojis by name, category, or meaning.  
- 🆕 **Latest Emojis:** Stay updated with new releases and trending emojis.  
- 📖 **Emoji Meanings:** Learn what every emoji really means.  
- 🌐 **Easy Sharing:** Copy and use emojis across all your favorite apps.  

## Why EmojiPedia?

Emojis are the language of emotions in the digital age. EmojiPedia makes it simple to **understand, find, and share** the perfect emoji in any conversation.

## Get Started

* Clone the repository:  
```bash
git clone https://github.com/YOUR_USERNAME/EmojiPedia.git
```
* Open terminal and run:
```bash
npm install unicode-emoji-json
```
* Open http://localhost:3000
* Enjoy EmojiPedia!
